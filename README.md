# Mentorship

This project contains the codes used for the mentorship I'm giving

## Getting Started

- Start the vitual env

```sh
$ python -m venv .
$ source ./bin/activate
```

- Installing dependencies

```sh
$ pip install -r requirements.txt
```

## Testing

```sh
$ pip install -r requirements.test.txt
$ pytest -q ./

# or

$ ptw
```
